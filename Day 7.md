## Personalized Claude Strategy

### Purpose
Use Claude to improve productivity, learning, research, and project development.

### Workflow
1. Define a clear goal.
2. Provide detailed context.
3. Ask Claude for a structured plan.
4. Review and refine responses.
5. Apply the output to projects.
6. Save useful prompts and workflows for future use.
