Day 20 – Face Puzzle Game 🎭🧩
Overview
As part of the 60 Days AB Talks Claude Challenge, I worked on creating a Face Puzzle Game that uses the device camera to capture a photo and convert it into an interactive puzzle.
Features
📷 Webcam access using getUserMedia()
🖼️ Live camera preview
📸 Photo capture functionality
🧩 Multiple difficulty levels (3×3, 4×4, 5×5)
🖱️ Drag-and-drop puzzle gameplay
📱 Touch support for mobile devices
⏱️ Timer tracking
🔢 Move counter
🏆 Leaderboard system
What I Learned
Browser Camera APIs
Image processing concepts
Dynamic UI generation
Drag-and-drop interactions
State management in JavaScript
Responsive web application design
